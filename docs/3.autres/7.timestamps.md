## Utilisation

La variable `{timestamp}` représente une date sous forme de timestamp Discord.

Exemple simplifié :
```
{timestamp}
```

::hint{ type="warning" }
  Utilisée seule, la variable `{timestamp}` renvoie uniquement une valeur numérique.
  Pour obtenir un affichage lisible, vous devez utiliser le [paramètre style](#paramètre-style).
::

::hint{ type="info" }
  Vous pouvez retrouver les autres variables existantes en [**`cliquant ici`**](/docs/autres/variables)
::

## Paramètre style

Le paramètre `style` permet de choisir le format d’affichage du timestamp.

### Variables

| Style | Exemple rendu | Description |
|-------|---------------|-------------|
| `t` | 14:32 | Heure courte |
| `T` | 14:32:18 | Heure complète |
| `d` | 25/01/2019 | Date courte |
| `D` | 25 janvier 2019 | Date longue |
| `f` | 25 janvier 2019 14:32 | Date et heure |
| `F` | vendredi 25 janvier 2019 14:32 | Date complète |
| `R` | Il y a 2 heures | Affichage relatif |

### Exemples d’utilisation

- **Date courte** : `{timestamp:style=d}`
- **Date et heure complète** : `{timestamp:style=F}`
- **Affichage relatif** : `{timestamp:style=R}`
