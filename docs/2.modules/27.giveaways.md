## Présentation

La fonctionnalité de giveaways vous permet d'organiser des tirages au sort et d'offrir des récompenses aux membres de votre serveur qui y participent.

![Exemple de giveaway avec une récompense personnalisée](../assets/giveaways/preview_giveaway.png)

Pour chaque giveaway, vous pourrez offrir l'une de ces récompenses :
- De l'expérience dans le système de [niveaux](/docs/modules/niveaux).
- De l'argent dans le système d'[économie](/docs/modules/economie).
- Un objet provenant du système d'[inventaire](/docs/modules/inventaire).
- Un rôle (temporaire ou non).
- Une récompense personnalisée.

## Interagir avec un giveaway

### Participer

Vous pouvez participer à un giveaway en cours en cliquant sur le bouton "Participer" en dessous des détails de celui-ci.

![Exemple de giveaway non terminé](../assets/giveaways/simple_giveaway.png)

Votre participation est alors prise en compte pour le tirage au sort à la fin du giveaway.

::hint{ type="info" }
  Nous ne proposons pas actuellement de fonctionnalités permettant à un membre de participer plusieurs fois à un giveaway.

  Tous les participants d'un giveaway ont donc les mêmes chances de victoires.
::

### Se désinscrire

Si vous avez décidé de rejoindre un giveaway, mais que vous ne souhaitez plus y participer avant la fin de celui-ci, vous le pouvez en cliquant de nouveau sur le bouton "Participer".

Une confirmation est alors demandée, cliquer sur le bouton "Se désinscrire" pour confirmer.

![Demande de confirmation de désinscription](../assets/giveaways/confirm_desinscription.png)

::hint{ type="warning" }
  Si une condition de collecte (rôle, argent ou objet) est définie sur le giveaway et que vous décidez de vous désinscrire, l'élément collecté ne vous sera pas restitué.
::

### Voir les participants

Si le créateur du giveaway l'a autorisé, vous pouvez voir la liste des participants en cliquant sur le bouton "Participants" du giveaway.

L'interface se découpe en page suivant le nombre de participants. Vous pouvez alors cliquer sur les boutons "Précédent", "Suivant" ou choisir le numéro de page avec le bouton "Page".

::hint{ type="info" }
  Si vous êtes administrateur, vous avez aussi accès au bouton "Retirer" qui permet de retirer la participation d'un membre. Plus d'informations dans la [partie dédiée](#gérer-les-participants).
::

## Accès la commande /giveaway

La commande \</giveaway> nécessite la permission **Administrateur** par défaut.

Néanmoins, il est possible de donner accès à certains rôles ou personnes l'accès à ces commandes indépendamment de leurs permissions sur le serveur.

## Créer un giveaway

Vous pouvez créer un giveaway en utilisant la commande \</giveaway créer>, en complétant les options de la commande :

| Option | Description | Obligatoire |
|--------|-------------|-------------|
| `récompense` | Le [type de récompense](#présentation). | ✅ Oui |
| `durée` | La durée du giveaway comprise entre une minute et un mois (ex: 2h, 7j) ou une date (ex: demain à 17h). | ✅ Oui |
| `gagnants` | Le nombre de gagnants du giveaway compris entre 1 et 10 (jusqu'à 50 pour les serveurs [premium](/premium) <:icon_premium_:1096140508625125417>). | ✅ Oui |
| `titre` | Le titre du giveaway. | ❌ Non |
| `salon` | Le salon textuel où sera envoyé le giveaway. | ❌ Non (par défaut le salon de la commande) |

::hint{ type="info" }
  Vous pouvez lancer jusqu'à 3 giveaways en simultané. Les serveurs [premium](/premium) <:icon_premium_:1096140508625125417> n'ont pas de limite.
::

Une fois la commande \</giveaway créer> exécutée, la récompense du giveaway vous est demandée :

![Formulaire de saisie pour une récompense de rôle](../assets/giveaways/modal_reward_role.png)

### Prévisualiser

Lorsque la récompense a été définie, une prévisualisation du giveaway vous est affichée, et vous permet de le configurer et le modifier avant sa publication.

![Interface de prévisualisation et de configuration d'un giveaway](../assets/giveaways/giveaway_config.png)

### Configurer les options

Depuis l'interface de prévisualisation du giveaway, vous pouvez configurer des options à l'aide des boutons présents ainsi que du sélecteur "Configurer des options avancées...".

L'ensemble des options sont détaillés dans leur [partie dédiée](#options-dun-giveaway).

### Publier le giveaway

Lorsque vous souhaitez le publier, cliquer sur le bouton "Lancer le giveaway".

::hint{ type="info" }
  Une fois le giveaway terminé, le créateur reçoit un message privé de <@DraftBot> l'informant du gagnant et lui rappelant la récompense.
::

## Options d'un giveaway

### Titre et description

Vous pouvez définir un titre qui est utilisé dans le message du giveaway ainsi que dans les messages envoyés aux membres de l'équipe du serveur (ex: message de fin, logs, etc.).

::hint{ type="info" }
  Si le titre n'est pas défini, le nom de la récompense est utilisé pour nommer le giveaway.
::

Vous pouvez également définir une description qui est utilisée dans le message du giveaway pour décrire votre récompense aux participants du giveaway.

![Giveaway avec un titre et un  personnalisée](../assets/giveaways/giveaway_description_title.png)

### Durée

La durée correspond au temps alloué aux membres pour participer à un giveaway.

Vous pouvez indiquer :
- soit une durée, comme `1h`, `7 jours` ou encore `2 semaines`.
- ou une date, comme `demain à 20h` ou `20/01 à 10h`.

::hint{ type="info" }
  La durée du giveaway doit être comprise entre une minute et un mois.
::

### Récompense

Vous avez la possibilité de sélectionner l'une de ces récompense pour un giveaway :

- De l'expérience dans le système de [niveaux](/docs/modules/niveaux).
- De l'argent dans le système d'[économie](/docs/modules/economie).
- Un objet provenant du système d'[inventaire](/docs/modules/inventaire).
- Un rôle (temporaire ou non).
- Une récompense personnalisée.

### Gagnants

Vous devez définir le nombre de gagnants pour votre récompense définie.

::hint{ type="info" }
  Vous pouvez choisir jusqu'à 10 gagnants. Les serveurs [premium](/premium) <:icon_premium_:1096140508625125417> peuvent aller jusqu'à 50 gagnants.
::

### Salon

Vous devez définir un salon textuel pour envoyer le giveaway lors de sa publication.

Vous avez le choix d'utiliser un salon existant ou de demander la création d'un salon dédiée au giveaway.

::hint{ type="info" }
  Par défaut, le salon utilisé est celui ou est executé la commande \</giveaway créer>.
::

#### Sélection d'un salon existant

Vous pouvez sélectionner un salon textuel du serveur existant pour l'envoi du giveaway à l'aide du sélecteur de salon.

![Sélection d'un salon existant](../assets/giveaways/exist_channel_choose.png)

::hint{ type="warning" }
  Le créateur du giveaway et <@DraftBot> doivent disposer des permissions **Envoyer des messages** et **Voir le salon** dans le salon.

  <@DraftBot> doit également disposer en plus de la permission **Intégrer des liens**.
::

#### Création d'un salon lors de la publication

::hint{ type="warning" }
  Vous devez disposer de la permission **Gérer les salons** pour créer un salon de cette manière.
::

Vous pouvez demander à <@DraftBot> la création d'un salon lors de la publication du giveaway, en cliquant sur le bouton "Nouveau salon dédié".

Vous pouvez alors définir le nom du salon ainsi que sa catégorie à l'aide du sélecteur et du bouton dédié.

![Création d'un salon lors de la publication du giveaway](../assets/giveaways/create_new_channel.png)

### Organisateur

La configuration d'un organisateur permet de signaler aux participants le membre donnant la récompense.

<image>

::hint{ type="warning" }
  L'organisateur doit être présent sur le serveur au moment de la configuration pour être sélectionné.
::

::hint{ type="info" }
  Le pseudo de l'organisateur est uniquement affiché sur le message et n'a pas d'autre effet sur le giveaway.
::

### Condition

Vous avez la possibilité de définir une condition d'accès pour participer au giveaway :

| Condition | Description |
|-----------|-------------|
| [Objet](/docs/modules/inventaire) | Posséder un objet dans son inventaire, avec possibilité de le retirer. |
| [Niveau](/docs/modules/niveaux) | Niveau compris entre un niveau minimum ou maximum. |
| [Argent](/docs/modules/economie) | Quantité d'argent compris entre un montant minimum ou maximum, avec possibilité de devoir payer la somme. |
| Rôles | Posséder (ou non) un ou plusieurs rôle(s), avec possibilité de les retirer. |

::hint{ type="warning" }
  En cas de retrait d'un élément (objet, niveau ou rôles) pour participer au giveaway, il ne sera pas restitué en cas de [désinscription](#se-désinscrire) ou si le membre ne gagne pas le giveaway.
::

### Images

Vous pouvez ajouter une vignette ainsi qu'une image principale à votre giveaway afin de l'égayer.

![Giveaway avec image et vignette (en haut à droite) configurées](../assets/giveaways/giveaway_image_thumbnail.png)

### Mentions

Vous pouvez définir des rôles à notifier lors de la publication du giveaway.

![Giveaway avec deux rôles mentionnés](../assets/giveaways/giveaway_with_mentions.png)

::hint{ type="info" }
  - Vous pouvez définir jusqu'à **10** rôles mentionnés.

  - Si vous ne possédez pas la permission **Mentionner @everyone, @here et tous les rôles** dans le salon du giveaway, vous ne pouvez mentionner que les rôles pouvant être mentionnés par tous les membres.
::

### Couleur

::hint{ type="info" }
  Cette option est réservée aux serveurs [premium](/premium) <:icon_premium_:1096140508625125417>.
::

Vous pouvez configurer la couleur latérale utilisée sur le message du giveaway afin de le correspondre à votre serveur ou à votre récompense.

![Giveaway avec une couleur personnalisée](../assets/giveaways/giveaway_custom_color.png)

### Bouton Participants

Le bouton Participants, par défaut activé, permet aux membres de [consulter la liste des participants](#voir-les-participants) d'un giveaway.

![Giveaway avec le bouton Participants activé](../assets/giveaways/simple_giveaway.png)

Néanmoins, il est possible de retirer ce bouton du giveaway. Les membres n'auront alors accès qu'au nombre de participants.

![Giveaway sans bouton Participants](../assets/giveaways/giveaway_no_participants_button.png)

::hint{ type="info" }
  Les administrateurs du serveur pourront toujours voir la liste des participants avec la commande \</giveaway participants>.
::

## Gérer un giveaway existant

### Arrêter un giveaway

Vous pouvez arrêter un giveaway en cours avant sa date de fin initiale avec la commande \</giveaway terminer>, en indiquant le message du giveaway ([identifiant](/docs/autres/recuperer-un-identifiant#identifiant-dun-message) ou son lien).

Le gagnant sera alors tiré au sort et signalé dans le salon du giveaway.

### Relancer un giveaway

Vous pouvez désigner un nouveau gagnant pour un giveaway terminé à l'aide de la commande \</giveaway relancer>, en indiquant le message du giveaway ([identifiant](/docs/autres/recuperer-un-identifiant#identifiant-dun-message) ou son lien).

::hint{ type="info" }
  Vous pouvez spécifier le gagnant à remplacer en saisissant l'option `gagnant` de la commande.
::

::hint{ type="warning" }
  Les récompenses gagnées ne sont pas retirées aux anciens gagnants lorsque vous en désignez un nouveau.
::

### Retirer un participant

Vous pouvez retirer un participant d'un giveaway en utilisant la commande \</giveaway participants> ou le bouton "Participants" du giveaway, si activé.

Vous pouvez alors sélectionner un participant à retirer à l'aide du bouton "Retirer un participant".

<image retirer un participant>

::hint{ type="info" }
  Le membre retiré ne sera pas averti de son retrait du giveaway.

  Néanmoins, il peut participer à nouveau au giveaway avant la fin de celui-ci.
::
